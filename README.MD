# Marketing Funnel & Conversion Analysis Dashboard

## Project Overview
This project focuses on analyzing user behavior and conversion patterns using Power BI. The dashboard tracks how users move through different stages of the marketing funnel and identifies major drop-off points that affect conversions.

The project provides business insights into user engagement, customer behavior, and funnel performance through interactive visualizations and KPI analysis.

---

## Objectives
- Analyze customer conversion funnel
- Identify major user drop-off stages
- Understand device-wise and gender-wise user distribution
- Monitor daily user activity trends
- Build an interactive business intelligence dashboard using Power BI
