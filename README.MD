# Marketing Funnel & Conversion Analysis Dashboard

## Project Overview
This project focuses on analyzing user behavior and conversion patterns using Power BI. The dashboard tracks user movement through different stages of the marketing funnel and identifies major drop-off points.

## Objectives
- Analyze user conversion funnel
- Identify drop-off stages
- Understand device and gender distribution
- Track daily user activity trends
- Build an interactive business intelligence dashboard

## Tools & Technologies
- Power BI
- Python
- Pandas
- Data Analytics
- Business Intelligence

## Dashboard Features
- KPI Cards
- Funnel Analysis
- Drop-off Analysis
- Device-wise User Distribution
- Gender-wise User Distribution
- Daily User Activity Trend
- Interactive Filters

## Key Insights
- Significant user drop-off occurs before purchase confirmation
- Mobile users dominate platform traffic
- Funnel conversion rate indicates optimization opportunity
- Search engagement remains relatively high
